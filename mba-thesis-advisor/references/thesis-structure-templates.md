# MBA/EMBA 论文结构模板

本文件提供 MBA/EMBA 论文最常见的三种结构模板。

---

### 模板 A：问题诊断+方案设计型（最常见，约占70%）

这种类型的论文以单个公司为案例，旨在诊断该公司在某一领域存在的问题，并提出系统性的解决方案。

```
第一章 绪论
  1.1 研究背景与意义
  1.2 研究问题界定
  1.3 研究内容与方法
  1.4 技术路线与论文结构

第二章 相关理论与文献综述
  2.1 [诊断理论] 理论基础
  2.2 [方案理论] 理论基础
  2.3 国内外相关研究现状
  2.4 文献评述与本研究的创新点

第三章 [公司名称] 概况与 [领域] 现状
  3.1 公司发展历程与业务概况
  3.2 公司所在行业环境分析
  3.3 公司 [领域] 发展现状

第四章 [公司名称] [领域] 问题诊断
  4.1 基于 [诊断理论] 的分析
  4.2 存在的核心问题识别
  4.3 问题成因深入分析

第五章 [公司名称] [领域] 优化方案设计
  5.1 优化方案的总体思路与目标
  5.2 基于 [方案理论] 的具体方案
  5.3 方案实施的保障措施

第六章 结论与展望
  6.1 研究结论总结
  6.2 研究不足之处
  6.3 未来研究展望
```

---

### 模板 B：调研分析型（约占20%）

这种类型的论文通过问卷调查或访谈，研究某个变量（如顾客满意度、员工离职意愿）的影响因素。

```
第一章 绪论

第二章 文献综述与研究假设
  2.1 核心概念界定
  2.2 相关理论回顾
  2.3 影响因素分析与研究假设提出
  2.4 研究模型构建

第三章 研究设计
  3.1 问卷设计与变量测量（采用成熟量表）
  3.2 抽样过程与样本特征
  3.3 数据分析方法

第四章 数据分析与假设检验
  4.1 问卷的信度与效度检验
  4.2 描述性统计分析
  4.3 相关分析与回归分析
  4.4 假设检验结果

第五章 研究结论与管理启示
  5.1 主要研究结论
  5.2 管理启示
  5.3 研究不足与展望
```

---

### 模板 C：多案例比较型（约占10%）

这种类型的论文通过比较 2-3 个公司的案例，总结共性规律或差异原因。

```
第一章 绪论

第二章 文献综述与分析框架构建
  2.1 相关理论回顾
  2.2 分析框架构建（提出要比较的维度）

第三章 研究设计
  3.1 案例选择与说明
  3.2 数据收集方法

第四章 案例分析
  4.1 案例 A 公司分析
  4.2 案例 B 公司分析
  4.3 案例 C 公司分析

第五章 跨案例比较与讨论
  5.1 基于分析框架的跨案例比较
  5.2 关键发现与命题提炼

第六章 结论与启示
```
