# MBA/EMBA 论文常用研究方法库

本文件详细介绍 MBA/EMBA 论文中最常用的几种研究方法，包括其适用场景、操作步骤和注意事项。

---

### 方法一：案例研究法 (Case Study)

- **一句话解释**: 对单一或少数几个现实情境中的特定对象（如一个企业、一个项目、一个决策）进行深入、全面的考察和分析。
- **适用场景**: 
  - 当研究问题是“怎么样(How)”和“为什么(Why)”时。
  - 当研究对象无法与现实情境分离开时。
  - 适用于探索性、描述性和解释性研究。
  - **绝大多数 MBA/EMBA 论文都以自己所在公司为案例进行研究。**
- **操作步骤**:
  1. **定义研究问题**: 明确你要通过案例研究回答什么问题。
  2. **选择案例**: 确定研究对象（通常是学生所在公司）。如果是多案例研究，需说明选择标准（如行业相似、规模不同等）。
  3. **设计数据收集方案**: 确定数据来源，主要包括：
     - **访谈**: 对关键人员（高管、部门负责人、核心员工）进行半结构化访谈。
     - **文档分析**: 收集公司年报、内部文件、会议纪要、媒体报道等。
     - **观察**: 对公司的工作流程、会议等进行实地观察。
  4. **数据收集**: 执行访谈、收集文档、进行观察。
  5. **数据分析**: 对收集到的数据进行编码、分类和主题分析，提炼关键发现。
  6. **撰写报告**: 按照论文结构，呈现案例背景、分析过程和研究结论。
- **注意事项**: 
  - **三角验证**: 尽量使用多种数据来源（如访谈+文档）相互印证，增加研究的信度。
  - **研究伦理**: 保护访谈对象的隐私，对敏感数据进行匿名化处理。

### 方法二：问卷调查法 (Survey Research)

- **一句话解释**: 通过设计标准化的问卷，向大量样本收集数据，以检验变量之间的关系或描述群体的特征。
- **适用场景**: 
  - 当需要量化研究变量之间的关系时（如检验假设）。
  - 当研究涉及较大范围的群体时（如某行业的用户、某公司的全体员工）。
  - 适用于描述性研究和解释性研究。
- **操作步骤**:
  1. **定义研究变量**: 明确你要测量的概念（如顾客满意度、员工敬业度、品牌形象等）。
  2. **设计问卷**: 
     - **借鉴成熟量表**: 优先使用学术界已经验证过的成熟量表，并在论文中注明来源。
     - **设计问题**: 问题应清晰、无歧义。多采用李克特五点或七点量表（如“非常不同意”到“非常同意”）。
     - **人口统计学信息**: 收集样本的基本信息（性别、年龄、职位等）。
  3. **确定抽样方案**: 定义目标人群，选择抽样方法（如方便抽样、滚雪球抽样）。
  4. **发放与回收问卷**: 通过在线问卷平台（如问卷星、Tencent问卷）发放。
  5. **数据分析**: 使用 SPSS, AMOS 等统计软件进行数据分析，主要包括：
     - **信度与效度检验**: 检验问卷的可靠性和有效性。
     - **描述性统计**: 描述样本的基本情况。
     - **相关分析/回归分析/结构方程模型**: 检验变量之间的关系。
  6. **撰写报告**: 呈现数据分析结果，并根据结果检验研究假设。
- **注意事项**: 
  - **样本量**: 确保有足够的有效样本量（通常建议至少 200-300 份）。
  - **预测试**: 在大规模发放前，找少量人员进行预测试，检查问卷是否存在问题。

### 方法三：访谈法 (Interview)

- **一句话解释**: 通过与研究对象进行直接的、有目的的对话来收集深入、详细的一手信息。
- **适用场景**: 
  - 当需要深入了解个人观点、动机、经历和感受时。
  - 适用于探索性研究，帮助发现问题和构建理论。
  - 常与案例研究法结合使用。
- **操作步骤**:
  1. **确定访谈对象**: 选择能够提供关键信息的个人（如高管、专家、核心员工）。
  2. **设计访谈提纲**: 准备一个半结构化的访谈提纲，包含需要询问的核心问题，但允许在访谈中灵活追问。
  3. **进行访谈**: 征得对方同意后进行录音。注意营造轻松的氛围，鼓励对方畅所欲言。
  4. **转录与整理**: 将访谈录音逐字转录为文本。
  5. **编码与分析**: 对访谈文本进行编码，提炼主题和模式。
- **注意事项**: 
  - **开放式问题**: 多使用“怎么样”、“为什么”、“请您详细谈谈”等开放式问题。
  - **避免诱导**: 保持中立，不要在问题中暗示你期望的答案。

### 方法四：文献分析法 (Literature Analysis)

- **一句话解释**: 通过系统性地收集、识别、分析和综述已有的文献（书籍、期刊、报告等），来构建研究的理论基础和发现研究空白。
- **适用场景**: 
  - 所有学术论文都必须使用的方法，是论文第二章“文献综述”的核心。
  - 也可作为主要研究方法，对某一主题的文献进行系统性梳理和元分析(Meta-Analysis)。
- **操作步骤**:
  1. **确定检索关键词**: 根据研究主题，确定中英文检索词。
  2. **选择数据库**: 使用知网(CNKI)、万方、Web of Science、Google Scholar 等学术数据库。
  3. **筛选与阅读文献**: 根据标题、摘要和相关性筛选文献，进行精读和泛读。
  4. **综述与述评**: 对文献进行分类、归纳和评述，而不是简单地罗列。要指出已有研究的贡献和不足，从而引出自己的研究价值。
- **注意事项**: 
  - **时效性**: 关注近 3-5 年的最新研究进展。
  - **权威性**: 优先引用高水平期刊和经典著作。

### 方法五：层次分析法 (Analytic Hierarchy Process, AHP)

- **一句话解释**: 一种将复杂的决策问题分解为目标、准则、方案等多个层次，并通过两两比较的方式确定各元素相对重要性权重的多准则决策方法。
- **适用场景**: 
  - 当需要在多个相互冲突的定性或定量准则下做出选择时（如供应商选择、方案评估、风险排序）。
- **操作步骤**:
  1. **建立层次结构模型**: 将问题分解为目标层、准则层和方案层。
  2. **构造判断矩阵**: 针对上一层次的某个元素，本层次的各个元素进行两两比较，使用 1-9 标度法确定其相对重要性。
  3. **计算权重与一致性检验**: 计算每个判断矩阵的权重向量，并进行一致性检验（CR < 0.1）。
  4. **计算组合权重**: 计算各层次元素的组合权重，得出方案的最终排序。
- **注意事项**: 
  - **专家打分**: 判断矩阵的构建通常需要邀请 5-10 位专家进行打分，以保证结果的可靠性。
  - **一致性检验**: 如果一致性检验不通过，需要调整判断矩阵。
